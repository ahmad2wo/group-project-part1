# User Stories - Online Restaurant Ordering System

- As a customer, I want to place an order as a guest so that I don’t need to create an account.
- As a customer, I want to browse the menu so that I can choose food easily.
- As a customer, I want to see food details (ingredients, calories, category) so that I can make informed choices.
- As a customer, I want to receive an order tracking number so that I can track my order.
- As a customer, I want to track my order status so that I know when it will be ready.
- As a customer, I want to choose delivery or takeout so that I can receive my food my way.
- As a customer, I want to pay using different methods so that payment is convenient.
- As a staff member, I want to view and manage orders so that operations run smoothly.
- As a chef, I want to see incoming orders so that I can prepare them efficiently.
- As a customer, I want to leave reviews and ratings so that I can share feedback.
- As a business owner, I want to create promotions and promo codes so that I can attract more customers.
- As a business owner, I want to analyze order data so that I can improve decisions.
