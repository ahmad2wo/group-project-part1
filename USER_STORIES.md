# User Stories

- As a user, I want to create an order so that I can place a request.
- As a user, I want to view all orders so that I can track them.
- As a user, I want to view a specific order so that I can see details.
- As a user, I want to update an order so that I can make changes.
- As a user, I want to delete an order so that I can cancel it.
- As a user, I want to manage order details so that I can customize my order.
